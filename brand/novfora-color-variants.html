NovFora — Color Tuning Sheet

Variants rendered on the real surfaces so you can pick precisely. Tell me e.g. “Ember → Film Amber, Dark → Warmer, Paper → keep.” The current brand‑board values are tagged CURRENT. AA notes are computed live.

1 · The Ember anchor (same hex in both modes)

Shown as a button (white label) on light parchment and on dark charcoal. “Btn AA” = white label on the fill (needs ≥4.5 for normal-size labels; ≥3.0 if labels are bold/large).

2 · Dark mode — “shadow on a photograph”

The warm-charcoal page surface. Ink + ember stay constant; only the background shifts.

3 · Light mode — “editorial print”

The matte-paper page surface. Pick how warm / how light the paper reads.

Reading the AA notes: green = passes WCAG AA, amber = passes only as bold/large or UI, red = fails as text. Borders are intentionally low-contrast (your “low-opacity warm” request) so they’re not scored here. Ink stays warm soft-black in light / film-grain off-white in dark across all options.
